## Getting Started

### Installation

1. Clone the repo

   ```sh
   git clone https://github.com/twilsonn/notedown.git
   ```

2. Install NPM packages

   ```sh
   npm install
   ```

3. Run development environment

   ```sh
   npm run dev
   ```
